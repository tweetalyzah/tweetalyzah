twitter.get_countries(tweets)[source]¶

Function used to get list of countries from which given tweets orginate. Countries are derived either from a ‘place’ parameter or by checking tweet’s coordinates with geopy. Return format is ready to be used by Google Visualization API.

twitter.get_hashtags()[source]¶

Function used to get list of most popular hashtags/trends with popularity associated with each hashtag. Collection is sorted by popularity.

twitter.get_languages()[source]¶

Function used to get list of languages used by friends. Friends’ languages are derived from twitter ‘lang’ parameter Return format is ready to be used by Google Visualization API.

twitter.get_recent_tweets()[source]¶

Function used to get tweets posted by friends in last 24 hours

twitter.in_previous_24h(date_str)[source]¶
Function used to to check whether date represented by given string
is within 24h period from current time
Parameters:date_str –
date_str: String with date. Preferred full date format
e.g. <Fri Jun 19 12:25:37 +0000 2015>
Returns:The boolean value. True for date within 24h, False otherwise.
